#  PlantGen AI

PlantGen AI is an open-source tool that helps generate plant product content using AI.

## Features

- Generate plant descriptions
- Create care guides (watering, sunlight, soil)
- Useful for plant shops and eCommerce websites
- Saves time for content creation

##  How it works

1. Enter a plant name
2. AI generates:
   - Description
   - Care instructions
3. Output can be added directly to a website

##  Example

Input:
Snake Plant

Output:
- Description: Snake plants are hardy indoor plants known for their upright leaves.
- Care Guide:
  - Water: Once every 2-3 weeks
  - Light: Indirect sunlight
  - Soil: Well-draining soil

##  Tech Stack

- Python
- AI APIs (Claude / OpenAI - optional)

##  Use Case

- Plant nurseries
- eCommerce websites
- Gardening blogs
  
##  Web App

This project includes a simple web interface where users can input a plant name and generate descriptions and care guides.

Run locally:

pip install flask  
python app.py
